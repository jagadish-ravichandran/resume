# Resume

This repository contains the LaTeX source for my professional resume.

## Files

* `main.tex` – Resume source file
* `Jagadish Ravichandran - Resume.pdf` – Generated resume

## Build

Compile the resume using Tectonic:

```bash
tectonic main.tex
```

The generated PDF will be created in the project directory.

## Requirements

* [Tectonic](https://tectonic-typesetting.github.io/)
